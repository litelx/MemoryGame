# MemoryGame
Memory game on web.\n
Two players or more (2-5 players) game with 3 different dificulties.\n
Easy difficulty: 5 pairs, 2X5 board.\n
Normal difficulty: 8 pairs, 4X4 board.
Difficult difficulty: 12 pairs, 4X6 board.

Technologies:
javascript
css
jquery
html5
bootstrap
